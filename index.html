Cadastro inicial do cliente

Este front é para captar os dados mínimos do cliente e enviar ao webhook do n8n, permitindo validar a criação no banco de dados e no Asaas.

Preencha a URL do webhook de teste para enviar os dados diretamente ao fluxo 1.
